JavaScript Tag & Terminologies Description Code Example
<script>Used to include the required JavaScript code in your HTML document.
<body>
<p id="showname"></p>
<script>
document.getElementById('showname').innerHTML='Peter';
</script>
</body>
<script src>Used to link the required JavaScript files in your HTML document.
<script src="script.js"></script>
varvar is a keyword used to declare variables.
var num1=10;
var num2=11;
var & Scopevar has functional scope, allowing variables to be accessed within a function only.
<script>
function show() {
  var name = 'Peter';
  document.getElementById('showname').innerHTML = name;
}
</script>
letUsed to declare block-scoped variables.
let num1=20;
let num2=21;
let & Scopelet has block scope, preventing redeclaration within the same scope.
<script>
{
  let emailId = 'test@example.com';
  document.getElementById('showemail').innerHTML = emailId;
}
</script>
constUsed to declare variables that cannot be reassigned.
const employeeId=120;
const & Scopeconst creates a constant whose value cannot be reassigned or redeclared.
<script>
{
  const employeeId = 120;
  document.getElementById('showEId').innerHTML = employeeId;
}
</script>
Arithmetic OperatorsPerform mathematical calculations like +, -, *, /, %.
let x=15, y=3;
console.log(x+y);
console.log(x-y);
console.log(x*y);
console.log(x/y);
console.log(x%y);
Comparison OperatorsCompare values and return true/false.
let a=5, b=7;
a==b;
a!=b;
a===b;
a>b;
Logical OperatorsCombine multiple conditions.
hasPermission && isMember;
hasPermission || isMember;
!hasPermission;
Assignment OperatorsAssign values to variables.
let x=10;
x+=5;
x-=5;
Unary OperatorsOperate on a single operand.
let count=5;
count++;
count--;
typeof OperatorReturns the data type of a variable.
typeof(42);
typeof("John");
if StatementExecutes a block if condition is true.
if(age>=18){console.log("Adult");}
else {console.log("Minor");}
else if StatementTests multiple conditions sequentially.
if(cond1){}
else if(cond2){}
else{}
Nested if elseTests multiple nested conditions.
if(temp>30){
  if(rain){...}
  else{...}
} else {
  if(rain){...}
  else{...}
}
switch StatementUsed for multiple conditional branches.
switch(day){
  case "Dec": ...; break;
  default: ...;
}
Ternary OperatorShort form of if...else.
let canVote = age>=18 ? "Yes":"No";
for loopRepeats code a set number of times.
for(let i=1;i<=5;i++){console.log(i);}
while loopRepeats as long as condition is true.
while(a<=50){...}
do...while loopExecutes at least once before checking the condition.
do{roll++;}while(roll<7);
Function Declaration and CallDefines and calls a reusable block of code.
function sayHello(){console.log("Hello");}
sayHello();
Non-Parameterized FunctionFunction without parameters.
function greet(){console.log("Hello World");}
greet();
Parameterized FunctionFunction that accepts arguments.
function add(a,b){return a+b;}
add(3,4);
Named FunctionFunction with a specific name.
const add=function(a,b){console.log(a+b);}
add(2,3);
IIFEFunction that runs immediately.
(function(){console.log("Welcome!");})();
Arrow FunctionConcise way to write functions.
const add=(a,b)=>a+b;
returnEnds a function and returns a value.
function multiply(msg){return msg;}
Function ClosureFunction remembers variables from its outer scope.
function outer(){
  const x="Hi";
  function inner(){console.log(x);}
  return inner;
}
outer()();
Function HoistingFunction declarations can be called before definition.
sayHello();
function sayHello(){console.log("Hello");}
Function Expression (no hoisting)Function expressions are not hoisted.
greet();
const greet=function(){console.log("Hi");}
addEventListenerAssigns a function to run on an event.
button.addEventListener('click',()=>{...});
onclick EventExecutes when element is clicked.
<button onclick="myFunction()">Click</button>
mouseover EventTriggered when mouse enters element.
element.addEventListener('mouseover',()=>{...});
mouseout EventTriggered when mouse leaves element.
element.addEventListener('mouseout',()=>{...});
keydown EventTriggered when key is pressed down.
input.onkeydown=(e)=>{console.log(e.key);}
change EventTriggered when input value changes.
input.onchange=()=>{console.log(input.value);}
onsubmit EventTriggered when form is submitted.
<form onsubmit="validateForm()">...</form>
