üñ±Ô∏è 5. Event Binding

TermDescriptionExample
Event An action detected by the browser (e.g., click, load, hover). Button click, page load
Event Binding Connecting an event to a JavaScript function that runs when it occurs. Run showAnswers() when a button is clicked.

Common Events

EventTriggerExample
onloadWhen the page finishes loading<body onload="init()">
onclickWhen an element is clicked<button onclick="run()">
onmouseoverMouse moves over element<p onmouseover="highlight()">
onfocusElement receives focus<input onfocus="highlightField()">
onkeyupKey released while typing<input onkeyup="checkInput()">
onsubmitForm submitted<form onsubmit="validateForm()">
onselectUser selects text<input onselect="textSelected()">
