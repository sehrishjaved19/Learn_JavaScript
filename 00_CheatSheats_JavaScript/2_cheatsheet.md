## ðŸ“˜ JavaScript DOM & BOM Terminologies

| **Terminology**                                  | **Description**                                                                                                                          | **Code Example**                                                                                                                                                                                                                                                                                                                                                                                                |
| ------------------------------------------------ | ---------------------------------------------------------------------------------------------------------------------------------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **`try { ... }` block**                          | The code that might generate an error is enclosed within a `try` block. This block helps monitor and handle potential errors gracefully. | `js {numberLines}  const obj = undefined;  try {      const propertyValue = obj.property; // Accessing undefined property      console.log("Property Value: " + propertyValue);      console.log("This message will be reached.");  } catch (error) {      console.error("An error occurred while accessing the property:", error.message);  }  console.log("Program continues after error handling."); `       |
| **`catch { ... }` block**                        | The `catch` block in JavaScript catches and handles errors that occur within a `try` block.                                              | `js {numberLines}  try {      const result = nondeclaredFunction(); // This will throw an error      console.log(result);  } catch (error) {      console.log('An error occurred:', error.message);  } `                                                                                                                                                                                                        |
| **`getElementById()` Method**                    | Used to access a specific HTML element within the DOM using its unique id attribute.                                                     | `html {numberLines}  <!DOCTYPE html>  <html>  <body>      <h1 id="main-heading">Welcome!</h1>      <p id="content">This is content.</p>      <script>          const headingElement = document.getElementById('main-heading');          console.log(headingElement);      </script>  </body>  </html> `                                                                                                         |
| **`getElementsByClassName()` Method**            | Accesses multiple elements that share the same class name.                                                                               | ``html {numberLines}  <!DOCTYPE html>  <html>  <body>      <p class="highlighted">Para 1</p>      <p class="highlighted">Para 2</p>      <script>          const highlighted = document.getElementsByClassName('highlighted');          for (let i = 0; i < highlighted.length; i++) {              highlighted[i].textContent = `Highlighted paragraph ${i+1}`;          }      </script>  </body>  </html> `` |
| **`querySelector()`**                            | Accesses a single HTML element using CSS selectors (.class, #id, tag) and returns the first match.                                       | `html {numberLines}  <p class="highlighted">Highlighted</p>  <script>      const el = document.querySelector('.highlighted');      console.log(el);  </script> `                                                                                                                                                                                                                                                |
| **`querySelectorAll()`**                         | Selects multiple elements using CSS-like selectors and returns a NodeList.                                                               | `html {numberLines}  <p class="highlighted">A</p>  <p class="highlighted">B</p>  <script>      const all = document.querySelectorAll('.highlighted');      console.log(all);  </script> `                                                                                                                                                                                                                       |
| **`textContent`**                                | Reads or modifies only the text content of an element (ignores HTML tags).                                                               | `html {numberLines}  <p id="para">Old text</p>  <script>      document.getElementById('para').textContent = 'Updated text';  </script> `                                                                                                                                                                                                                                                                        |
| **`setAttribute()` Method**                      | Changes or adds attributes (like `src`, `href`, or `id`).                                                                                | `html {numberLines}  <img id="pic" src="old.jpg">  <script>      document.getElementById('pic').setAttribute('src', 'new.jpg');  </script> `                                                                                                                                                                                                                                                                    |
| **Adding Elements**                              | Dynamically adds new HTML elements using `createElement()` and `appendChild()`.                                                          | `html {numberLines}  <ul id="list"><li>A</li></ul>  <script>      let li = document.createElement('li');      li.textContent = 'B';      document.getElementById('list').appendChild(li);  </script> `                                                                                                                                                                                                          |
| **`cloneNode()` Method**                         | Creates a copy of an element. `true` clones its children too.                                                                            | `html {numberLines}  <ul id="list"><li>A</li></ul>  <script>      let item = document.querySelector('li');      document.getElementById('list').appendChild(item.cloneNode(true));  </script> `                                                                                                                                                                                                                 |
| **`window` Object**                              | Represents the browser window and provides global BOM properties and methods.                                                            | `js {numberLines}  window.alert('Hi');  window.open('https://example.com');  console.log(window.location.href); `                                                                                                                                                                                                                                                                                               |
| **`navigator` Object**                           | Gives information about the browser and system.                                                                                          | `js {numberLines}  console.log(navigator.appName);  console.log(navigator.userAgent); `                                                                                                                                                                                                                                                                                                                         |
| **`screen` Object**                              | Contains details about the userâ€™s screen dimensions.                                                                                     | `js {numberLines}  console.log(screen.width, screen.height); `                                                                                                                                                                                                                                                                                                                                                  |
| **`history` Object**                             | Allows navigation through the browserâ€™s session history.                                                                                 | `js {numberLines}  history.back();  history.forward(); `                                                                                                                                                                                                                                                                                                                                                        |
| **`location` Object**                            | Provides or changes the current URL.                                                                                                     | `js {numberLines}  console.log(location.href);  location.href = 'https://example.com'; `                                                                                                                                                                                                                                                                                                                        |
| **`firstElementChild()` & `lastElementChild()`** | Accesses the first and last child elements of a parent.                                                                                  | `html {numberLines}  <div id="p"><p>A</p><p>B</p></div>  <script>      let d = document.getElementById('p');      console.log(d.firstElementChild.textContent);  </script> `                                                                                                                                                                                                                                    |
| **Container Element**                            | Queries elements inside a specific container to limit the search scope.                                                                  | `html {numberLines}  <div id="box"><p class="x">A</p></div>  <script>      const el = document.getElementById('box').querySelector('.x');  </script> `                                                                                                                                                                                                                                                          |
| **`element.style.property = value`**             | Changes inline styles via JS (CSS properties in camelCase).                                                                              | `html {numberLines}  <button id="b">Click</button>  <script>      let b = document.getElementById('b');      b.style.color = 'white';      b.style.backgroundColor = 'blue';  </script> `                                                                                                                                                                                                                       |
| **`element.classList`**                          | Adds, removes, or toggles CSS classes easily.                                                                                            | `html {numberLines}  <div id="d" class="active"></div>  <button id="btn">Toggle</button>  <script>      btn.onclick = () => d.classList.toggle('active');  </script> `                                                                                                                                                                                                                                          |
| **`element.setAttribute('style', '...')`**       | Directly sets inline CSS through a style string.                                                                                         | `html {numberLines}  <p id="p" style="color:red;">Hi</p>  <script>      p.setAttribute('style', 'color:blue;font-size:18px');  </script> `                                                                                                                                                                                                                                                                      |
| **`element.style.cssText`**                      | Sets all inline CSS at once, replacing previous styles.                                                                                  | `html {numberLines}  <p id="p">Hi</p>  <script>      p.style.cssText = 'color:green;font-weight:bold;';  </script> `                                                                                                                                                                                                                                                                                            |
| **`element.style.setProperty()`**                | Sets a specific CSS property, with optional priority like `important`.                                                                   | `html {numberLines}  <h1 id="h">Hi</h1>  <script>      h.style.setProperty('color', 'violet', 'important');  </script> `                                                                                                                                                                                                                                                                                        |
| **`element.style.removeProperty()`**             | Removes a CSS property from inline style.                                                                                                | `html {numberLines}  <p id="p" style="color:blue;">Hi</p>  <script>      p.style.removeProperty('color');  </script> `                                                                                                                                                                                                                                                                                          |
