üìñ Glossary: JavaScript Programming for Web Applications

Term Definition
AJAX ‚ÄúAsynchronous JavaScript and XML‚Äù that encompasses more than asynchronous server calls through JavaScript and XML. It is not a programming language but rather a programming concept combining HTML, JavaScript, and CSS to make web pages dynamic. JSON is now more common than XML.
Anonymous Functions Functions without names, often used as arguments or immediately invoked after definition.
Array A data structure for storing ordered elements accessible by index, starting at 0. Arrays can grow or shrink dynamically.
Classes Blueprints for creating objects with shared properties and methods.
Client-Side Script Programs that run in the browser, typically embedded using the <script> tag. They can validate forms or dynamically create content.
Document Objects Represents the HTML page loaded in the browser and provides access to all its elements via JavaScript.
DOM The Document Object Model, a programming interface that lets JavaScript dynamically access and modify webpage elements.
Element Nodes All HTML tags within the DOM structure.
Element Objects Base class objects in a document that hold shared properties and methods for all HTML elements.
Event Any action the user or browser performs that JavaScript can respond to, such as a click or form submission.
Event Binding The process of linking a function to run when a particular event occurs.
Event Handlers Functions that specify what to do when a specific event happens, such as onclick for button clicks.
Extend Keyword used in class declarations to create a subclass from another class.
Functions Reusable blocks of code that perform specific tasks and may return values.
IIFE Immediately Invoked Function Expression ‚Äî executes right after its definition and cannot be reused later.
Nodes The fundamental units in the DOM tree ‚Äî everything is a node (elements, text, comments).
Objects Instances of classes representing data and functionality grouped together.
Prototypes Used to add or inherit properties/methods among JavaScript objects.
Script Code that extends HTML functionality, handles events, or dynamically manipulates the DOM.
Self-Executing Functions Functions that execute automatically once defined; commonly used for initialization.
Text Nodes Nodes that hold the actual text content within an element‚Äôs start and end tags.
this Keyword referring to the current object instance depending on how the function is called.

üìò JavaScript DOM & BOM Terminologies

Terminology Description Code Example
try { ... } The code that might generate an error is enclosed within a try block. This block helps monitor and handle potential errors gracefully. const obj = undefined; try { console.log(obj.name); } catch (err) { console.error(err.message); }
catch (error) Catches and handles any errors thrown inside the try block. try { ... } catch (error) { console.log(error.message); }
finally Executes code regardless of whether an error occurred or not. try { ... } catch { ... } finally { console.log("Done"); }
throw Manually throws an error to be caught by catch. throw new Error("Custom error message");
window The global object representing the browser window. window.alert("Hello");
document Represents the HTML document loaded in the browser. document.title = "New Title";
navigator Contains information about the browser and OS. navigator.userAgent
screen Provides information about the user's screen dimensions. screen.width
location Represents the current URL of the document. location.href
history Gives access to the browser‚Äôs session history. history.back(); history.forward();
alert() Displays a simple alert dialog with a message. alert("Welcome");
confirm() Displays a dialog with OK and Cancel buttons. confirm("Are you sure?");
prompt() Displays a dialog asking for user input. prompt("Enter your name:");

üí° JavaScript DOM Access Methods

Method Description Example
document.getElementById() Selects the first element with the specified ID. document.getElementById("main")
document.getElementsByClassName() Returns all elements with the specified class name (as an HTMLCollection). document.getElementsByClassName("item")[0]
document.getElementsByTagName() Returns all elements with the specified tag name. document.getElementsByTagName("div")
document.querySelector() Selects the first element that matches a CSS selector. document.querySelector(".container p")
document.querySelectorAll() Returns all elements matching the given CSS selector (NodeList). document.querySelectorAll("p.note")
document.forms Returns all form elements in the document. document.forms[0].elements["username"].value
document.images Returns all <img> elements in the document. document.images.length
document.links Returns all hyperlinks (<a> tags with href). document.links[0].href
document.scripts Returns all <script> elements. document.scripts
document.title Access or modify the document title. document.title = "New Title"

Method Description Example
createElement() Creates a new HTML element. document.createElement("div")

removeAttribute() Removes an attribute. element.removeAttribute("class")

üß† JavaScript Event Handling Cheat Sheet

Event Type Description Example
onclick Fires when an element is clicked. button.onclick = function() { alert("Clicked!"); }
onload Triggered when a page or image is loaded. window.onload = function() { console.log("Page Loaded"); }
onchange Occurs when the value of an input element changes. input.onchange = () => console.log("Changed");
onmouseover Fires when mouse pointer moves over an element. div.onmouseover = () => div.style.color = "blue";
onmouseout Fires when mouse pointer leaves the element. div.onmouseout = () => div.style.color = "black";
onkeydown Triggered when a key is pressed down. document.onkeydown = e => console.log(e.key);
onkeyup Triggered when a key is released. document.onkeyup = e => console.log(e.key);
addEventListener() Registers an event handler for a specified event. element.addEventListener("click", myFunction)
removeEventListener() Removes an event handler. element.removeEventListener("click", myFunction)
