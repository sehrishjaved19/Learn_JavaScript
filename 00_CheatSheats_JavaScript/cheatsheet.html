Class or Method Description Example
appendChild() After creating an element, this function places it in the appropriate location within the document. Takes the element to append as the only parameter. 
<head>
<script>
  function addPara() {
    var newPara = document.createElement("p");
    var newText = document.createTextNode("Hello World!");
    newPara.appendChild(newText);
    document.body.appendChild(newPara);
  }
</script>
</head>
<body onload="addPara()"></body>
Arrays Created by declaring elements in [ ]. Arrays use zero-based indexing to access elements. 
const Beatles = ["Ringo", "Paul", "George", "John"];
// Beatles[0] â†’ "Ringo"
Date() Returns the current local date and time if no parameters are provided. You can pass parameters to set a specific date/time. 
var newDate = new Date("2021-1-17 13:15:30");
var newDate = new Date(2021, 0, 17); // Jan 17, 2021
document.createElement() Creates an element with the given tag name. Can be placed using methods like appendChild() or insertBefore(). 
var p = document.createElement("p");
document.createTextNode() Creates a text node with the provided string. 
var txt = document.createTextNode("Hello World!");
document.getElementById() Returns an element with the specified ID. 
<div id="div1"></div>
<script>
document.getElementById("div1").innerHTML = "Hello World!";
</script>
document.getElementsByTagName() Returns a NodeList of all elements with the specified tag name. 
var tags = document.getElementsByTagName("p");
document.write() Writes HTML or JS output directly into the document (overwrites existing content). 
document.write("Hello World");
element.getAttribute() Returns the value of a specified attribute. 
var color = div1.getAttribute("style");
element.innerHTML Gets or sets the inner HTML content of an element. 
div1.innerHTML = "<p>Updated</p>";
element.removeAttribute() Removes a specific attribute from an element. 
div1.removeAttribute("style");
element.setAttribute() Sets or updates an attribute and its value. 
document.getElementById("img").setAttribute("src", "pic.jpg");
element.style Accesses or modifies inline CSS styles. 
div1.style.color = "red";
Error Objects Contain properties like message and name. Custom errors can be thrown using throw. 
throw new Error("Invalid value");
History Objects Contains URLs visited in a session; allows navigation through go() and other methods. 
history.go(-2);
insertBefore() Inserts a node before a specific child node. 
ul.insertBefore(newItem, ul.firstChild);
Location Objects Contains information about the current page URL. 
let host = location.hostname;
Navigator Objects Represents the browser (user agent). 
var browser = navigator.appName;
onload() Runs a function after the page fully loads. 
window.onload = function() { init(); };
replaceChild() Replaces an existing child node with a new one. 
list.replaceChild(newItem, oldItem);

Window Objects The global object that represents the browser window. 
window.open("http://www.w3schools.com");
window.open() Opens a new browser window or tab. 
window.open("https://www.ibm.com", "myWin", "width=600,height=800");
window.scrollTo() Scrolls the document to specified coordinates. 
window.scrollTo(20, 200);
Wrapper Objects Convert primitive values to objects (e.g., String, Number). 
typeof "abc"; // string
typeof new String("abc"); // object
