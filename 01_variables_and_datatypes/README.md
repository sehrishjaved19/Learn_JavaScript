# Variables and Data Types in JavaScript

This folder introduces the foundational concepts of variables and data types in JavaScript. It contains example scripts and notes to help you understand how to declare, name, and use variables, as well as the different data types available in JavaScript.

## Contents

- `01_hello_world.js` — Basic hello world example
- `02_variable_declaration.js` — How to declare variables using `var`, `let`, and `const`
- `03_variable_scope.js` — Explanation of variable scope (global, local, block)
- `04_variable_naming.js` — Rules and best practices for naming variables
- `05_data_types.js` — Overview of JavaScript data types (string, number, boolean, null, undefined, object, symbol)
- `NOTES.md` — Additional notes and explanations
- `README.md` — This documentation file

## Key Concepts

1. **Variable Declaration:** Learn the difference between `var`, `let`, and `const`.
2. **Scope:** Understand how scope affects variable accessibility.
3. **Naming Conventions:** Follow best practices for naming variables.
4. **Data Types:** Explore primitive and reference data types in JavaScript.

## How to Use

1. Review each `.js` file for code examples and explanations.
2. Refer to `NOTES.md` for deeper insights and summary points.
3. Experiment by running the scripts in your browser or Node.js environment.
