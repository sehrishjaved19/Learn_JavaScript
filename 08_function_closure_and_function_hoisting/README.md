# Function Closure and Function Hoisting in JavaScript

This folder explores two important concepts in JavaScript: closures and function hoisting. It provides examples and explanations to help you understand how these features work and how to use them effectively.

## Contents

- `01_function_closure.js` — Demonstrates the concept of closures in JavaScript
- `02_function_hoisting.js` — Explains function hoisting and its effects
- `NOTES.md` — Additional notes and explanations
- `README.md` — This documentation file

## Key Concepts

1. **Closures:** Learn how functions can capture and remember their lexical environment
2. **Function Hoisting:** Understand how function declarations are hoisted and how this affects code execution

## How to Use

1. Review each file for code examples and explanations
2. Refer to `NOTES.md` for summary points and extra details
3. Experiment by running the scripts in your browser or Node.js environment
