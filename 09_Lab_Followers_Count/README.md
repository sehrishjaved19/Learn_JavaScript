# Lab: Followers Count in JavaScript

This folder contains a lab exercise for managing and displaying a followers count using JavaScript. The included files provide an interactive web page, supporting styles, and JavaScript logic for updating and showing the number of followers.

## Contents

- `followers_count.html` — Interactive web page for followers count
- `followers_count.css` — Stylesheet for the lab interface
- `followers_count.js` — JavaScript code for managing followers count
- `README.md` — This documentation file

## Lab Objectives

1. Practice using JavaScript to update and display dynamic values
2. Implement functions to increase, decrease, or set the followers count
3. Create an interactive user interface for input and output
4. Apply DOM manipulation and event handling

## How to Use

1. Open `followers_count.html` in your browser to use the followers count tool
2. Use the interface to change the number of followers
3. View the updated count displayed on the page
4. Review and modify `followers_count.js` to experiment with different logic or features
