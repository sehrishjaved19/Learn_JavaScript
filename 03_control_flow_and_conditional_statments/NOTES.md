# Control Flow & Conditional Statements

## Control Flow:
**Control flow refers to the order in which statments are executed in JavaScript program.**

## Conditional Statements:
**Conditionla Statments often refered to as decision making statments, are used to mange this flow based on specific conditions.

### Types of Conditional Statments in JavaScript:

* if Statement
* else if Statement
* else Statement
* nested if else Statement
* switch Statement
* Ternary Operator

---
## If Statement
**The if Statment is used to execute a block of code if a specified condition is true. If the condition is false, the code block is skipped.**

---
## Else if Statement
**The Else If Statement allows you to test multiple conditions sequentially, especially when you have more than two possible outcomes.The Else Statement is used to specify a block of code to be executed if the condition in the If Statement is false.**

---

## Nestef if else Statement
**Nested If Else Statements are a common programming construct in JavaScript and many other programming languages. They allow you to test multiple conditions and execute different blocks of code based on the results of those conditions.**

---

## Switch Statement
**The Switch Statement allows you to compare a value against multiple possible case values and execute code based on the first matching case.**

---

## Ternary Operator
**The Ternary Operator is a concise way to write conditional statements, such as If Else Condition.**
