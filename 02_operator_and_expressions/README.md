# Operators and Expressions in JavaScript

This folder provides examples and explanations of the various operators and expressions used in JavaScript. Understanding operators is essential for performing calculations, making decisions, and manipulating data in your programs.

## Contents

- `01_arithmetic_operators.js` — Demonstrates arithmetic operations (addition, subtraction, multiplication, division, modulus)
- `02_comparison_operator.js` — Shows how to compare values using comparison operators
- `03_logical_operator.js` — Explains logical operators (AND, OR, NOT)
- `04_assignment_operator.js` — Covers assignment operators for setting and updating variable values
- `05_unary_operator.js` — Details unary operators (increment, decrement, typeof, etc.)
- `06_typeof_operator.js` — Explains the `typeof` operator for checking data types
- `07_expressions.js` — Provides examples of different types of expressions
- `NOTES.md` — Additional notes and explanations

## Key Concepts

1. **Arithmetic Operators:** Perform mathematical calculations
2. **Comparison Operators:** Compare values and return boolean results
3. **Logical Operators:** Combine or invert boolean values
4. **Assignment Operators:** Assign and update variable values
5. **Unary Operators:** Operate on a single operand
6. **Expressions:** Combine values, variables, and operators to produce results

## How to Use

- Review each `.js` file for code examples and explanations
- Refer to `NOTES.md` for summary points and extra details
- Experiment by running the scripts in your browser or Node.js environment
