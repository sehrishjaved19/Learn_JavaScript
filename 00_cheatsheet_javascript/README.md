# JavaScript Cheat Sheets

This folder contains quick reference materials and cheat sheets for JavaScript programming concepts. It serves as a handy resource for both beginners and experienced developers to quickly look up common JavaScript syntax, methods, and best practices.

## Contents

- `index.html` - The main HTML file that displays the cheat sheets
- `style.css` - Styling for the cheat sheet presentation
- `README.md` - This documentation file

## Topics Covered

1. Variables and Data Types
2. Operators and Expressions
3. Control Flow and Conditional Statements
4. Loops and Iterations
5. Functions
6. Objects and Classes
7. Arrays and Array Methods
8. DOM Manipulation
9. Events
10. Async JavaScript (Promises, Callbacks)
11. Error Handling
12. ES6+ Features

## How to Use

1. Open the `index.html` file in your web browser to view the cheat sheets
2. Use the navigation to find the specific topic you're looking for
3. Each section contains code examples and explanations
