### ðŸ“˜ Cheatsheet: Introduction to JavaScript Development

| **JavaScript Tag & Terminologies**    | **Description**                                                                           | **Code Example**                                                                                                           |   |                               |
| ------------------------------------- | ----------------------------------------------------------------------------------------- | -------------------------------------------------------------------------------------------------------------------------- | - | ----------------------------- |
| `<script>`                            | Used to include the required JavaScript code in your HTML document.                       | `html <body> <p id="showname"></p> <script> document.getElementById('showname').innerHTML='Peter'; </script> </body> `     |   |                               |
| `<script src>`                        | Used to link the required JavaScript files in your HTML document.                         | `html <script src="script.js"></script> `                                                                                  |   |                               |
| `var`                                 | `var` is a keyword used to declare variables.                                             | `js var num1=10; var num2=11; `                                                                                            |   |                               |
| `var & Scope`                         | `var` has **functional scope**, allowing variables to be accessed within a function only. | `html <script> function show() { var name = 'Peter'; document.getElementById('showname').innerHTML = name; } </script> `   |   |                               |
| `let`                                 | Used to declare block-scoped variables.                                                   | `js let num1=20; let num2=21; `                                                                                            |   |                               |
| `let & Scope`                         | `let` has **block scope**, preventing redeclaration within the same scope.                | `html <script> { let emailId = 'test@example.com'; document.getElementById('showemail').innerHTML = emailId; } </script> ` |   |                               |
| `const`                               | Used to declare variables that cannot be reassigned.                                      | `js const employeeId=120; `                                                                                                |   |                               |
| `const & Scope`                       | `const` creates a constant whose value cannot be reassigned or redeclared.                | `html <script> { const employeeId = 120; document.getElementById('showeEId').innerHTML = employeeId; } </script> `         |   |                               |
| **Arithmetic Operators**              | Perform mathematical calculations like +, -, *, /, %.                                     | `js let x=15, y=3; console.log(x+y); console.log(x-y); console.log(x*y); console.log(x/y); console.log(x%y); `             |   |                               |
| **Comparison Operators**              | Compare values and return true/false.                                                     | `js let a=5, b=7; a==b; a!=b; a===b; a>b; `                                                                                |   |                               |
| **Logical Operators**                 | Combine multiple conditions.                                                              | ```js hasPermission && isMember; hasPermission                                                                             |   | isMember; !hasPermission; ``` |
| **Assignment Operators**              | Assign values to variables.                                                               | `js let x=10; x+=5; x-=5; `                                                                                                |   |                               |
| **Unary Operators**                   | Operate on a single operand.                                                              | `js let count=5; count++; count--; `                                                                                       |   |                               |
| **typeof Operator**                   | Returns the data type of a variable.                                                      | `js typeof(42); typeof("John"); `                                                                                          |   |                               |
| **if Statement**                      | Executes a block if condition is true.                                                    | `js if(age>=18){console.log("Adult");} else {console.log("Minor");} `                                                      |   |                               |
| **else if Statement**                 | Tests multiple conditions sequentially.                                                   | `js if(cond1){} else if(cond2){} else{} `                                                                                  |   |                               |
| **Nested if else**                    | Tests multiple nested conditions.                                                         | `js if(temp>30){ if(rain){...} else{...} } else { if(rain){...} else{...} } `                                              |   |                               |
| **switch Statement**                  | Used for multiple conditional branches.                                                   | `js switch(day){case"Dec":...;break;default:...;} `                                                                        |   |                               |
| **Ternary Operator**                  | Short form of if...else.                                                                  | `js let canVote = age>=18 ? "Yes":"No"; `                                                                                  |   |                               |
| **for loop**                          | Repeats code a set number of times.                                                       | `js for(let i=1;i<=5;i++){console.log(i);} `                                                                               |   |                               |
| **while loop**                        | Repeats as long as condition is true.                                                     | `js while(a<=50){...} `                                                                                                    |   |                               |
| **do...while loop**                   | Executes at least once before checking the condition.                                     | `js do{roll++;}while(roll<7); `                                                                                            |   |                               |
| **Function Declaration and Call**     | Defines and calls a reusable block of code.                                               | `js function sayHello(){console.log("Hello");} sayHello(); `                                                               |   |                               |
| **Non-Parameterized Function**        | Function without parameters.                                                              | `js function greet(){console.log("Hello World");} greet(); `                                                               |   |                               |
| **Parameterized Function**            | Function that accepts arguments.                                                          | `js function add(a,b){return a+b;} add(3,4); `                                                                             |   |                               |
| **Named Function**                    | Function with a specific name.                                                            | `js const add=function(a,b){console.log(a+b);} add(2,3); `                                                                 |   |                               |
| **IIFE**                              | Function that runs immediately.                                                           | `js (function(){console.log("Welcome!");})(); `                                                                            |   |                               |
| **Arrow Function**                    | Concise way to write functions.                                                           | `js const add=(a,b)=>a+b; `                                                                                                |   |                               |
| **return**                            | Ends a function and returns a value.                                                      | `js function multiply(msg){return msg;} `                                                                                  |   |                               |
| **Function Closure**                  | Function remembers variables from its outer scope.                                        | `js function outer(){const x="Hi"; function inner(){console.log(x);} return inner;} outer()(); `                           |   |                               |
| **Function Hoisting**                 | Function declarations can be called before definition.                                    | `js sayHello(); function sayHello(){console.log("Hello");} `                                                               |   |                               |
| **Function Expression (no hoisting)** | Function expressions are **not hoisted**.                                                 | `js greet(); const greet=function(){console.log("Hi");} `                                                                  |   |                               |
| **addEventListener**                  | Assigns a function to run on an event.                                                    | `js button.addEventListener('click',()=>{...}); `                                                                          |   |                               |
| **onclick Event**                     | Executes when element is clicked.                                                         | `html <button onclick="myFunction()">Click</button> `                                                                      |   |                               |
| **mouseover Event**                   | Triggered when mouse enters element.                                                      | `js element.addEventListener('mouseover',()=>{...}); `                                                                     |   |                               |
| **mouseout Event**                    | Triggered when mouse leaves element.                                                      | `js element.addEventListener('mouseout',()=>{...}); `                                                                      |   |                               |
| **keydown Event**                     | Triggered when key is pressed down.                                                       | `js input.onkeydown=(e)=>{console.log(e.key);} `                                                                           |   |                               |
| **change Event**                      | Triggered when input value changes.                                                       | `js input.onchange=()=>{console.log(input.value);} `                                                                       |   |                               |
| **onsubmit Event**                    | Triggered when form is submitted.                                                         | `html <form onsubmit="validateForm()">...</form> `                                                                         |   |                               |
