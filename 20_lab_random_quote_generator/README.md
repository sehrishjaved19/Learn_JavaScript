# Random Quote Generator

A simple, elegant web application that generates random inspirational quotes at the click of a button.

## Features

- Clean and modern user interface
- One-click quote generation
- Responsive design that works on all devices
- Smooth animations and transitions
- Elegant typography with decorative quote marks

## Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla)

## Project Structure

```
random_quote_generator/
│
├── index.html      # Main HTML structure
├── style.css       # Styling and animations
├── script.js       # Quote generation logic
└── README.md       # Project documentation
```

## How It Works

1. The application loads with an initial empty quote display area
2. When the user clicks the "Get Quote" button:
   - JavaScript generates a random quote from the predefined collection
   - The quote is displayed in the main quote area with smooth transition
   - Decorative quote marks frame the text for visual appeal

## Styling Features

- Gradient background for visual interest
- Card-style quote display with shadow effects
- Responsive button with hover and active states
- Mobile-friendly design with appropriate text sizing
- Decorative quote marks using CSS pseudo-elements

## Usage

1. Clone or download the repository
2. Open `index.html` in a modern web browser
3. Click the "Get Quote" button to generate random quotes
